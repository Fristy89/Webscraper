Voor een klant van Visser & Van Baars in de regio van Utrecht ben ik op zoek naar een senior SQL server DBA. 
Wat moet je meebrengen als Senior SQL server DBA?
Meer dan 5 jaar ervaring als SQL Server DBA (2014, 2016)
Kennis van Agile/scrum
In deze opdracht ga je als SQL server database beheerder nauw samenwerken met het ontwikkelteam. Hierdoor zijn je communicatieve vaardigheden zeer belangrijk. Tijdens de opdracht ga je je vooral bezig houden met het beheer van de database waarin Troubleshooten & Performance Tuning de basis vormt.
Kun jij jezelf vinden in het gevraagde profiel en ben je per direct of op korte termijn beschikbaar? Laat het mij dan direct weten door een korte motivatie te schrijven en een cv te uploaden (in Word). Indien dit voldoende strookt met de gevraagde expertise en ervaring neem ik zo snel mogelijk contact met je op.
